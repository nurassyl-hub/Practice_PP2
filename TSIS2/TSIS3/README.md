# TSIS3 Racer Game

Advanced Racer game using Pygame.

## Assets

Put these files into the assets folder:

- Player.png
- Enemy.png
- coin.png
- AnimatedStreet.png
- background.wav
- crash.wav

## Features

- Main Menu
- Settings screen
- Leaderboard screen
- Game Over screen
- Username entry
- Dynamic traffic cars
- Road obstacles: barriers, oil, potholes
- Road events: moving barrier, speed bump, nitro strip
- Power-ups: Nitro, Shield, Repair
- Difficulty scaling
- Persistent leaderboard saved to leaderboard.json
- Persistent settings saved to settings.json

## Controls

- Arrow keys or WASD = move car
- Mouse = menu buttons

## Run

```bash
pip install pygame
python main.py
```
