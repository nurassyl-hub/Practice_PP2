# TSIS2 Paint Application

Extended Paint application using only Pygame.

## Features

- Pencil freehand drawing
- Straight line with live preview
- Rectangle
- Circle
- Square
- Right triangle
- Equilateral triangle
- Rhombus
- Eraser
- Color picker
- Brush size: 2px, 5px, 10px
- Flood fill tool
- Text tool
- Ctrl+S saves canvas as timestamped PNG

## Keyboard Shortcuts

- P = Pencil
- L = Line
- R = Rectangle
- C = Circle
- S = Square
- A = Right triangle
- Q = Equilateral triangle
- H = Rhombus
- F = Fill
- T = Text
- E = Eraser
- 1 = Small brush
- 2 = Medium brush
- 3 = Large brush
- Ctrl+S = Save canvas

## Run

```bash
pip install pygame
python paint.py
```
